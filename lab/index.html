Hatch Text

솔리드 텍스트 위에 ::after로 사선 레이어를 덮는 두 레이어 구조.
solid color + background-clip: text (hatch overlay)

variants

Hatching 01 — 흰 사선 / -45deg / 빠름 Striped 02 — 굵은 사선 / -60deg / 느림 Crossed 03 — 교차 격자 / ±45deg / 반대 방향 Scanned 04 — 수직 스캔 / 90deg / 아래로

how it works

/* HTML: data-text에 같은 텍스트를 넣어준다 */
<span class="hatch" data-text="Skills">Skills</span>

/* 레이어 1: 솔리드 텍스트 */
.hatch {
  position: relative;
  display: inline-block;
  color: #c05540; /* 솔리드 색상 그대로 보임 */
}

/* 레이어 2: 사선 패턴을 같은 텍스트 모양으로 덮음 */
.hatch::after {
  content: attr(data-text); /* 같은 텍스트 */
  position: absolute;
  inset: 0;
  background-image: repeating-linear-gradient(
    -45deg,
    transparent           0px,
    transparent           5px,
    rgba(255,255,255,0.45) 5px,
    rgba(255,255,255,0.45) 8px
  );
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent; /* 사선만 보이게 */
  animation: hatch-slide 1.2s linear infinite;
}

@keyframes hatch-slide {
  from { background-position: 0 0; }
  to   { background-position: 40px 0; }
}
